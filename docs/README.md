# 魔法基地突襲 — 設計文件總覽

> 一句話定位:一款可愛、低暴力、節奏快的魔法冒險休閒遊戲。玩家經營自己的法師聖所、布置防禦、突襲別人的聖所;先以 Poki 輕量版驗證市場,再用同一顆核心長成深度版。

這份 `docs/` 收錄專案目前所有定案文件,依「先讀什麼」排序。

## 文件關係

- **設計統整** 是玩法的根;§10「元素=機制」取代了早期的相剋三角。
- **法杖／零件系統規格** 是 §10 的延伸(加層,不取代)。
- **技術棧定案** 只管怎麼蓋,不重述玩法。
- **Demo 階段計畫** 與 **master backlog** 把設計與技術對到可執行的進度。

## 核心紀律(貫穿所有文件)

- **一套核心,兩種包裝**:資料驅動的確定性戰鬥核,深度版是「加東西」而非「重寫」。
- **每一階只加一樣、過關才往上爬**;Demo 0 是總閘門。
- **互乘而非疊加**:不做純數值、不做嚴格更強、不做無上限 RNG。
- **不要兩個殼並行開發**:先把 Poki 輕量殼做成 vertical slice 上線測市場。
